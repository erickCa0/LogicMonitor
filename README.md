# LogicMonitor

Welcome to my collection of custom PowerShell tools, automation, and integrations for the **LogicMonitor Envision Platform**.

This repository serves as a home for the utilities, modules, reporting tools, and proof-of-concepts I've built while working with LogicMonitor. Some projects solve common operational challenges, while others explore new ways to automate, extend, or integrate the platform beyond what's available out of the box.

Whether you're looking for a reporting utility, an API helper, or inspiration for your own automation, I hope you'll find something useful here.

## Built Around the Logic.Monitor Module plus more.

Most projects leverage the excellent Logic.Monitor PowerShell Module and other APIs that LogicMonitor that can extend it with additional functionality where needed. When native cmdlets don't exist—or when a workflow can be simplified—I've built custom solutions to fill the gaps.

## Why This Repository Exists

After building numerous one-off scripts over the years, I wanted a central place to organize, improve, and share them. Rather than letting useful code disappear into customer engagements or personal archives, this repository gives those tools a permanent home.

My goals are simple:

- Build practical tools that solve real-world problems.
- Share reusable code instead of reinventing it.
- Explore new capabilities of the LogicMonitor platform.
- Continuously refine and improve existing solutions.

## Project Status

This is an active, evolving repository. New tools are added regularly, existing projects are refactored as the platform evolves, and documentation is expanded over time.

Not every project is production-ready, but each represents an idea, solution, or workflow that I found valuable enough to preserve.
